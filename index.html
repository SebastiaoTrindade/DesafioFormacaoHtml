Tags explicadas nas aulas

Tags de título

As tags de título possuem valor semântico, variando entre seis níveis hierárquicos.

  • H1 - Título de maior valor
  • H2
  • H3
  • H4
  • H5
  • H6 - Título de menor valor

Tags de texto

As tags de texto definem textos, estilos de fonte, parágrafos, spans, quebras de linhas, etc.

  • p - Principal tag de texto, compõe um parágrafo.
  • mark - Usada para representar um texto em destaque.
  • small - Utilizado para reduzir tamanho de fonte em um , definido no texto.
  • i - Transforma a parte do texto aplicado em itálico.
  • u - Transforma a parte do texto aplicado em sublinhado - underline.
  • strong - Transforma a parte do texto aplicado em negrito.
  • hr - Ela coloca uma linha horizontal aonde é aplicado.
  • sub - É utilizado para escrever um texto com letras menores com alinhamento abaixo do conteudo, exemplo. h2o.
  • sup - É utilizado para escrever um texto com letras menores com alinhamento acima do conteudo, exemplo. 22.
  • blockquote - É utilizado para inserir uma citação, exemplo. Html - Linguagem de Marcação.
  • font - É utilizado para configurar o tipo, cor e tamanho das letras de um texto na página Html.
  • del - É utilizado para indicar que um texto é suprimido. Ex. Este texto é suprimido.
  • abbr - É utilizado para inserir uma sigla em uma página Html.

Tags de multimídia

As tags de multimídia servem para incluir imagens, vídeos, áudios, iframes e outros tipos de conteúdo multimídia.

  • img - Serve para incluir uma imagem ao seu texto, utiliza o atributo src=” ” onde deve ser digitado o local em que a imagem se encontra.

Tags de Lista

Esta tag serve para criar listas ordenadas e não ordenadas.

Minha lista ordenada:

  1. item 1
  2. item 2
  3. item 3

Minha lista não ordenada:

  • item 1
  • item 2
  • item 3
